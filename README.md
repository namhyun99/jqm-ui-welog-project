## jqm-ui-welog-project
- jQuery Mobile을 활용한 웹앱제작

## 개발환경 및 개발툴
- OS : Microsoft Windows11 64bit
- DBMS : Oracle11gXE
- Language : HTML, CSS, JQuery, JQueryMobile, Java 1.8
- Editor : eclipse, sqlDeveloper

## 개발기간
- 2022.07.25.～ 2022.08.02 
- 글쓰기 기능 추가 예정..

## 기능설명
- 로그인, 회원가입, 회원정보 수정 기능
- 아이디 찾기, 비밀번호 찾기 기능 
- 로컬스토리지를 이용한 다크모드 구현
- html편집기 적용
